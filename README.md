# SevenMinuteWorkout
The 7 minutes workout consists of only 12 exercises to be done for 30 seconds, with 10-second breaks between each exercise. All you need is a chair and a wall. Repeat 2-3 circuits depending on how much time you have. Make it your first choice at home or at the office.
